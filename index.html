Set up RuneProfile

This page only shows stats after the player has a public profile on RuneProfile and has synced from RuneLite.

  1. Run Old School RuneScape through RuneLite.
  2. Open Plugin Hub, search for RuneProfile, and install the plugin.
  3. Open the RuneProfile sidebar panel in RuneLite, sign in (or create an account), and use Sync so your skills, quests, collection log, and clan info upload to the site.
  4. Open this member link again using the player’s exact OSRS login name (the name used to log into the game).

Clan members who have already synced will appear in the roster; until then, lookups here will show this message instead of stats.

      Terpinheimer RuneLite plugin

      User guide: what the plugin does, how to open it, and how to connect it to this site, Wise Old Man, Discord, and the live map.

      What it does

      Terpinheimer adds a clan hub panel in RuneLite’s sidebar. It can show Wise Old Man competitions, website calendar status, post your position to the live map, and send optional Discord notifications when things happen in game.

      Install

      Get the plugin (Plugin Hub or jar) and detailed setup from Terpinheimer Discord.

      Opening the plugin

      1. Open RuneLite’s sidebar (plugin list).
      2. Find Terpinheimer and click its icon to open the panel.

      Home tab

      • Announcements — short clan message. Only Owner / Deputy Owner (Jagex clan rank) can save changes to the text in RuneLite config.
      • Quick links — Discord, website, Wise Old Man, live map, etc., if you filled those URLs under plugin Links.
      • Skill of the Week / Boss of the Week — from your Wise Old Man group’s competitions; shows leaders, countdowns, and links to the competition on the web.
      • Web Events — reads your clan’s calendar summary API so upcoming or active events from this site can appear in the client.
      • Clan Event tracker — opened from Home (attendance card), with present/late thresholds and copy behavior controlled in config.

      Tabs in the sidebar

      • Always shown — Home, SOTW, BOTW, Web Events.
      • Conditional — Group tab appears only while you are in a RuneLite party (Party plugin/passphrase), and shows shared party loot totals + split.
      • Attendance is intentionally opened from Home and is not duplicated as a bottom tab.

      Wise Old Man

      • Set your clan’s Wise Old Man group ID under General in the plugin config (0 uses the default Terpinheimer group).
      • Refresh interval controls how often competition data updates.
      • Update WOM profile on logout (optional) — on logout or world hop, if you gained enough XP or leveled, the plugin can refresh your stats on Wise Old Man.

      RuneProfile

      Optional Update RuneProfile on logout uses the same XP/level idea as WOM and helps keep RuneProfile in sync so member pages on this site stay accurate. This sync is built into Terpinheimer (no extra RuneProfile plugin required for skills/quests on logout). Full collection log depth can still benefit from the official RuneProfile plugin.

      Live clan map

      • Enable Live clan map → Enable live map API in the plugin.
      • Under Links, set Live map API base URL to this site’s origin only (e.g. https://your-site.com) — no path.
      • Set Live map API shared key to match LIVE_MAP_SECRET on the server (same as in your host’s environment).
      • The client POSTs your position to {base}/post on a schedule so the website map can show you.
      • You can hide your pin during events, skip sending in the Wilderness, or change how often updates are sent.

      Discord notifications

      All optional. Paste one webhook URL under Discord → Webhook, then enable categories you want:

      • Loot — NPC/player drops (filters, minimum value, screenshots).
      • Clue scrolls — treasure trail rewards.
      • Pets — pet messages.
      • Collection log — new collection log entries.
      • Levels — level-ups (intervals, min level, virtual/combat levels, screenshots).
      • Progression — deaths, quests, combat achievements.
      • Coffer donations — optional donation notices through the same webhook flow.

      Leave the webhook empty and leaves sends off if you only want the sidebar — no Discord spam.

      Clan event attendance

      Optional tracker for who was present at a clan event (Jagex clan chat / area-based, similar in spirit to other attendance plugins). Open it from the plugin Home area (e.g. clan event tracker). Adjust present/late thresholds and copy options under Clan event attendance in config.

      Links in config

      Under Links, fill Discord invite, website, Wise Old Man group, live map page, clan calendar page, calendar summary API URL, and live map base URL + key as needed. Empty fields usually hide the related button in the panel.

      Quick checklist

      1. Install Terpinheimer from the hub or your jar.
      2. General → confirm Wise Old Man group ID.
      3. Links → set website and Discord at minimum.
      4. Links → set clan calendar page + summary API so Web Events works.
      5. Optional: enable live map + base URL + shared key.
      6. Optional: join a RuneLite party and enable Party loot (Group tab).
      7. Optional: Discord webhook + turn on the notification types you want.
